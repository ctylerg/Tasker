# Tasker

The *MEAN* way to keep yourself on task!

<h2> **Now GET TO WORK!**

Tasker is built using the following technologies:
- MongoDB
- Express.js
- Angular.js
- Node.js

##API Access
To access tasks, please use the *Tasker* RESTFul API!

```bash
GET     /api/tasks
GET     /api/tasks/:id
POST    /api/tasks
PUT     /api/tasks/:id
DELETE  /api/tasks/:id
```
